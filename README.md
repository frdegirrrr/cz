# CZ

Веб-утилита шифрования текста по русскому алфавиту. Один автономный файл
`index.html` — работает офлайн в браузере, без установки и без зависимостей.

## Режимы

- **Классический** — шифр Цезаря со сдвигом-ключом (1…32), сохраняет регистр.
- **Усложнённый** — нумеруются только русские буквы; каждая 3-я буква сдвигается
  на +2 вперёд, остальные — на −1 назад. Ключа нет, расшифровка однозначна.
  Текст приводится к строчным, переносы строк заменяются пробелом.

## Запуск

Открыть `index.html` двойным кликом в любом браузере.

## Публикация (GitHub Pages)

Файл лежит в корне как `index.html`, поэтому после включения GitHub Pages сайт
доступен по адресу `https://<логин>.github.io/<репозиторий>/`.
