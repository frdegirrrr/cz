# CZ

Веб-утилита шифрования с двумя режимами. Один автономный файл `index.html` —
работает в браузере, без установки и без зависимостей.

## Режим «Грузинский шифр»

В русском и грузинском (мхедрули) алфавитах по 33 буквы. Шифр комбинированный:
сдвиги по буквам, затем перевод на грузинский.

- **Зашифровать:** нумеруются только русские буквы; каждая 3-я сдвигается на +2
  вперёд, остальные — на −1 назад (по модулю 33), затем результат переводится на
  грузинский (буква №i русского → буква №i грузинского).
- **Расшифровать:** обратно — грузинский → русский, затем обратные сдвиги
  (каждая 3-я буква −2, остальные +1).
- Пробелы, цифры и знаки проходят без изменений и не сбивают счёт букв.
- Ввод приводится к строчным, переносы строк заменяются пробелом; в грузинском
  нет заглавных, поэтому расшифровка выдаёт строчный русский текст.

## Режим «Ключи (RSA)»

Настоящее асимметричное шифрование через встроенный в браузер Web Crypto API,
без внешних библиотек. Гибридная схема: RSA-OAEP 2048 бит шифрует случайный
AES-GCM 256 ключ, которым шифруется сам текст (поэтому работает текст любой длины).

- **Сгенерировать пару** — создаёт публичный и приватный ключ (формат PEM).
- **Зашифровать** — публичным ключом получателя.
- **Расшифровать** — своим приватным ключом.
- Ключи генерируются в браузере и нигде не сохраняются; приватный ключ хранить
  в секрете. Требуется защищённый контекст (HTTPS) — на сайте GitHub Pages есть.

## Запуск

Открыть `index.html` двойным кликом в любом браузере.

## Публикация (GitHub Pages)

Файл лежит в корне как `index.html`, поэтому после включения GitHub Pages сайт
доступен по адресу `https://<логин>.github.io/<репозиторий>/`.
