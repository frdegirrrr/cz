# CZ — веб-утилита шифра (спецификация)

Дата: 2026-07-22

## Назначение
Рабочий инструмент для шифрования русского текста через грузинский алфавит.
Одна автономная HTML-страница, работает офлайн в браузере, без Python и без
внешних зависимостей.

## Логика шифра
В русском и грузинском (мхедрули) алфавитах по 33 буквы. Шифр комбинированный:
сдвиги по позиции буквы, затем перевод на грузинский.

- Русский: `абвгдеёжзийклмнопрстуфхцчшщъыьэюя` (33).
- Грузинский: `აბგდევზთიკლმნოპჟრსტუფქღყშჩცძწჭხჯჰ` (33).
- **Зашифровать:** нумеруются только русские буквы; каждая 3-я → +2 вперёд,
  остальные → −1 назад (по модулю 33); затем сдвинутая буква переводится в
  грузинскую по позиции (буква №i русского → буква №i грузинского).
- **Расшифровать:** грузинский → русский по позиции, затем обратные сдвиги
  (каждая 3-я буква → −2, остальные → +1).
- Пробелы, цифры, знаки проходят без изменений и не сбивают счёт букв.
- Ввод нормализуется: строчные + переносы строк → пробел. В грузинском заглавных
  нет, поэтому расшифровка выдаёт строчный русский текст.

## Режим «Ключи (RSA)» (добавлен 2026-07-22)
Настоящее асимметричное шифрование через Web Crypto API (без внешних библиотек),
новым режимом рядом с грузинским. Переключатель вверху: `Грузинский шифр` /
`Ключи (RSA)`.

- Гибридная схема: RSA-OAEP 2048 бит (SHA-256) шифрует случайный AES-GCM 256 ключ,
  которым шифруется текст. Поэтому работает текст любой длины.
- Формат результата: `base64(encAesKey).base64(iv).base64(ciphertext)`.
- Ключи в PEM (SPKI для публичного, PKCS8 для приватного).
- UI: кнопка «Сгенерировать пару» + поля публичного/приватного ключа; поле «Ключ
  для операции» (публичный при шифровании, приватный при расшифровке).
- Ключи генерируются в браузере, нигде не сохраняются. Требуется защищённый
  контекст (HTTPS); на GitHub Pages выполняется.
- Это стандартная криптография, но инструмент самодельный и без аудита — не для
  критичных секретов.

## Интерфейс
1. Переключатель режимов: `Грузинский шифр` / `Ключи (RSA)`.
2. (RSA) Карточка «Ваши ключи»: генерация + публичный/приватный ключ.
3. Textarea ввода текста; (RSA) поле «Ключ для операции».
4. Кнопки **Зашифровать** / **Расшифровать**.
5. Вывод результата + кнопка «Копировать».
6. Заголовок `CZ`, адаптив под светлую/тёмную тему браузера.

## История
Проект начинался как шифр Цезаря (классический сдвиг + усложнённый режим со
сдвигами по позиции буквы). По решению пользователя эти режимы заменены единым
русско-грузинским шифром-подстановкой.

## Вне рамок (YAGNI)
Сдвиг/ключ поверх перевода, другие алфавиты, сохранение файлов, история операций.

## Файлы
`index.html` — единственный файл, HTML + встроенные CSS/JS.
