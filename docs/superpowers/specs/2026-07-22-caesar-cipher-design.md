# Код Цезаря — веб-утилита (спецификация)

Дата: 2026-07-22

## Назначение
Рабочий инструмент для шифрования/расшифрования текста шифром Цезаря по
русскому алфавиту. Одна автономная HTML-страница, работает офлайн в браузере,
без Python и без внешних зависимостей.

## Логика шифра
- Алфавит: 33 русские буквы `абвгдеёжзийклмнопрстуфхцчшщъыьэюя`,
  буква **ё на своём месте** (после е). Сдвиг по модулю 33.
- Регистр сохраняется (`А`→`Г`, `а`→`г` при сдвиге 3).
- Всё, что не русская буква (цифры, пробелы, знаки, латиница), проходит без изменений.
- Расшифровка — сдвиг в обратную сторону на тот же ключ.

## Интерфейс
1. Textarea ввода текста.
2. Поле сдвига `1…32` (число + ползунок, синхронизированы).
3. Кнопки: **Зашифровать**, **Расшифровать**, **Взлом**.
4. Вывод:
   - шифр/дешифр — один результат + кнопка «Копировать»;
   - «Взлом» — 33 строки «сдвиг N → текст» для ручного поиска читаемого варианта.

## Оформление
Минимализм, адаптив под светлую/тёмную тему браузера (`prefers-color-scheme`).

## Режим «Усложнённый» (добавлен 2026-07-22)
Переключатель режимов вверху: `Классический` / `Усложнённый`.

Классический режим — без изменений (сдвиг, три кнопки, взлом, регистр сохраняется).

Усложнённый режим:
- Нумеруются **только русские буквы** (1, 2, 3…); пробелы/цифры/знаки пропускаются
  и не сбивают счёт.
- Каждая **3-я буква → +2 вперёд**, все остальные → **−1 назад**.
- Ключа нет — расшифровка однозначна (обратные сдвиги по той же нумерации).
- Ввод нормализуется: приводится к строчным, переносы строк → пробел.
- В этом режиме поле «Сдвиг» и кнопка «Взлом» скрыты; доступны только
  «Зашифровать» / «Расшифровать».
- Позиция буквы одинакова в открытом и шифрованном тексте (буква→буква), поэтому
  расшифровка по номеру позиции корректна.

## Вне рамок (YAGNI)
Авто-выбор «самого читаемого» варианта, латиница/другие алфавиты, сохранение
файлов, история операций.

## Файлы
`index.html` — единственный файл, HTML + встроенные CSS/JS.
